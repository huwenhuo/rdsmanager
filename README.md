# rdsmanager

Manage your R workspace, one variable at a time. Automatic saving and loading of R variables with RDS files.

[![R](https://img.shields.io/badge/R-%3E%3D%203.6-blue.svg)](https://www.r-project.org/)

## Features

- Save individual variables from your R workspace as `.rds` files.
- Maintain a table mapping variables to filenames.
- Automatically load variables back into the same names.
- Lightweight, minimal, and easy to integrate into any workflow.
